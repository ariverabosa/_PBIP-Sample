# Power BI Program Analysis

Business Intelligence portfolio project built with Power BI, SQL, and DAX to analyze academic program performance, enrollment trends, and key institutional indicators.

## Objective

Develop an end-to-end Power BI reporting solution that demonstrates:

- Data modeling and semantic model organization
- DAX measure development
- SQL-based data preparation
- Dashboard design for business users
- PBIP project structure and version control practices
- Reporting workflow organization

## Tools & Technologies

- Power BI Desktop
- DAX
- SQL
- Power Query
- PBIP
- Data modeling
- Star schema concepts
- GitHub version control

## Key Features

- Semantic model organization
- PBIP project structure for source control compatibility

## Business Questions

Examples of questions addressed in this project:

- Which academic programs show changes in student enrollment?
- How do program trends evolve over time?
- Which programs or program groups require additional review?
- How can reporting improve visibility into academic program performance?
